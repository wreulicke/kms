## Usage

```sh
$ kms -h

  kms [flags]
  kms [command]

Available Commands:
  decrypt     decrypt
  encrypt     encrypt
  help        Help about any command

  -h, --help             help for kms

Use "kms [command] --help" for more information about a command.
```
